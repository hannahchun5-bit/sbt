About Soulbound Tokens

A Soulbound Token (SBT) is a non-transferable, non-fungible token that represents identity, credentials, or commitments. Unlike traditional NFTs, SBTs cannot be bought, sold, or transferred.

Key Properties

  • Non-transferable
  • Non-fungible
  • Identity-based
  • Publicly verifiable
  • Immutable record
